# Golf-Performance-Analysis-Dashboard
An Excel-based golf analytics project designed to track scoring performance, evaluate consistency, and identify the statistical relationships between key golf metrics and overall score.

---

## Project Overview

This project analyzes historical golf performance data across multiple rounds to uncover trends and performance drivers using data visualization, probability analysis, and correlation metrics.

The dashboard was built entirely in Microsoft Excel and focuses on transforming raw golf statistics into actionable insights through an interactive analytics-style layout.

---

## Key Features

- Score trend analysis over time
- Histogram visualization of score distribution
- Scatterplot analysis between putts and score
- Probability calculations for breaking target scores
- Correlation analysis between:
  - Putts and Score
  - GIR and Score
  - Fairways Hit and Score

---

## Metrics Tracked

- Total Score
- Score vs Par
- Fairways Hit
- Greens in Regulation (GIR)
- Putts
- Penalties

---

## Statistical Analysis

The project uses Excel statistical functions to analyze relationships between variables:

- `CORREL()` to measure relationships between performance metrics
- `COUNTIF()` for probability calculations
- `AVERAGE()`, `MIN()`, and `MAX()` for KPI summaries

Example insights:
- Higher putts are strongly associated with worse scores
- Increased GIR is associated with improved scoring performance
- Score consistency and improvement trends can be visualized over time
